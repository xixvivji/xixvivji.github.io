🐶 유기견 입양·사후관리 올인원 프로세스

기간: 2026.01.15 ~ 2026.02.09

역할: 인프라 · 백엔드 (팀 6인)

📌 프로젝트 개요

유기견 입양부터 사후관리까지 이어지는 올인원 서비스입니다. OpenVidu 기반 화상 상담 기능을 포함하여 입양 전 상담 및 이후 관리 프로세스를 지원합니다.

🚀 핵심 기여

  • CI/CD 구축 – Jenkins 기반 파이프라인 구성으로 배포 자동화 및 재현성 확보
  • 배포 인프라 안정화 – Docker Compose · Nginx 설정 정비로 외부 접근 및 라우팅 안정화
  • 화상 상담 안정화 – OpenVidu 프록시/CORS/인증서 이슈 해결
  • 모니터링 도입 – Prometheus · Grafana 구성으로 운영 가시성 확보
  • 스토리지 연동 – AWS S3 업로드 설정 및 검증 API 구현
  • 백엔드 품질 개선 – Spring Security · Gradle 설정 정비로 서비스 안정성 향상

🛠 문제 해결 사례

1️⃣ 화상 상담 연결 불안정

  • 원인: 프록시 경로/포트/CORS/인증서 설정 충돌
  • 조치: Nginx · Docker Compose · OpenVidu 설정 재정비
  • 결과: 실서비스 환경에서 화상 상담 연결 안정화

2️⃣ 배포 실패 반복

  • 원인: 환경별 설정 불일치
  • 조치: Jenkins 파이프라인 구조 개선 및 환경 변수 체계화
  • 결과: 배포 재현성 확보 및 실패율 감소

🧰 기술 스택

CI/CD: Jenkins
Infra: Docker, Docker Compose, Nginx, OpenVidu
Backend: Spring Boot, Gradle
Cloud: AWS S3
Monitoring: Prometheus, Grafana
